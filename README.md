# Mi pagina web

Web estatica lista para publicar con GitHub Pages.

## Como publicarla en GitHub

1. Crea un repositorio nuevo en GitHub.
2. Sube estos archivos al repositorio:
   - `index.html`
   - `styles.css`
   - `script.js`
3. En GitHub, entra en **Settings**.
4. Abre **Pages**.
5. En **Build and deployment**, elige:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
6. Guarda los cambios.

GitHub te dara una URL parecida a:

```text
https://tu-usuario.github.io/nombre-del-repositorio/
```

## Que cambiar primero

- En `index.html`, cambia "Mi Web" por el nombre de tu pagina.
- Cambia `tu-correo@example.com` por tu correo real.
- Sustituye los textos de proyectos por tu contenido.
